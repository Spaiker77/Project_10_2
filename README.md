# Проект Маскировки Номеров

Этот проект реализует функции для маскировки номеров банковских карт и счетов, а также предоставляет возможность фильтрации данных по состоянию.

## Структура проекта

Проект состоит из следующих модулей:

1. masks: 
   - get_mask_card_number(card_number: str) -> str: Функция для маскировки номера банковской карты. Возвращает замаскированный номер карты.
   - get_mask_account(account_number: str) -> str: Функция для маскировки номера банковского счета. Возвращает замаскированный номер счета.

2. widget:
   - mask_account_card(info: str) -> str: Функция, которая принимает строку, содержащую тип и номер карты или счета, и возвращает строку с замаскированным номером. Использует разные типы маскировки для карт и счетов.

3. processing:
   - filter_by_state(data: list, state: str = 'EXECUTED') -> list: Функция, которая принимает список словарей и опционально значение для ключа state. Возвращает новый список словарей, содержащий только те словари, у которых ключ state соответствует указанному значению.

4. generators:
  - filter_by_currency(transaktions: Any, code: str) -> Generator: Генераторная функция, которая возвращает транзакции с заданной валютой. Принимает список транзакций и код валюты, фильтруя только те транзакции, которые соответствуют указанной валюте.
  - transaction_descriptions(transactions: Any) -> Generator: Генератор, который принимает список словарей с транзакциями и возвращает описание каждой операции по очереди.
  - card_number_generator(start: int, stop: int) -> Generator: Генератор, который выдает номера банковских карт в формате XXXX XXXX XXXX XXXX, где X — цифра номера карты. Принимает начальное и конечное значение диапазона для генерации номеров карт.

5. utils:
   - read_transactions(file_path: str) -> List[Dict[str, Any]]: Функция, которая Читает транзакции из JSON-файла и возвращает их в виде списка словарей. Если файл не найден, пустой или содержит не список, возвращает пустой список.

6. transaction_reader:
   - read_csv_transactions: Функция, которая читает транзакций из CSV файла
   - read_excel_transactions: Функция, которая читает транзакций из Excel файла

7. external_api:
   - convert_transaction_amount(transaction: Dict[str, Any]): Функция, которая конвертирует сумму транзакции в рубли. Для USD и EUR использует внешний API.

8. decorators:
   - log(filename=None): Декоратор, который будет автоматически логировать начало и конец выполнения функции, а также ее результаты или возникшие ошибки.

9. main:
   - load_json(file: str): Загружает JSON файл
   - load_csv(file: str): Загружает CSV файл
   - filter_by_status: Фильтрует по статусу
   - sort_by_date: Сортирует по дате
   - filter_rub: Оставляет только рублевые транзакции
   - print_transactions: Выводит транзакции

10. operations:
   - filter_by_description: Фильтрует транзакции по строке в описании
   - count_by_category: Считает операции по категориям
